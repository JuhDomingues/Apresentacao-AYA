# AYA - Plataforma de Agentes Inteligentes | SEIDOR

AYA é a plataforma revolucionária de agentes inteligentes totalmente customizáveis da SEIDOR. Desbloqueie o potencial da IA para transformar seu negócio.

## Tecnologias

Este projeto é construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Desenvolvimento

### Pré-requisitos

- Node.js e npm - [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Como executar localmente

```sh
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

O servidor estará disponível em `http://localhost:8080`

### Scripts disponíveis

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria build de produção
- `npm run build:dev` - Cria build de desenvolvimento
- `npm run lint` - Executa o linter
- `npm run preview` - Preview do build de produção
